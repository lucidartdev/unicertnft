# UniCert NFT — On-Chain Education Credential Platform

**UniCert NFT** is a blockchain-based platform for issuing, managing, and verifying educational credentials as NFTs. Universities or authorized institutions can mint NFT certificates for students, while students can securely share and verify their achievements on-chain.

## Technologies Used

- **Frontend:** Next.js 14 (App Router) + TypeScript  
- **Styling:** Tailwind CSS + shadcn/ui  
- **Wallet Integration:** WalletConnect / Reown AppKit  
- **Smart Contract:** Solidity ERC721 NFT  
- **Blockchain:** Base Mainnet

## Features

- Mint NFT certificates for students with metadata including:
  - Student Name
  - Course Name
  - Grade
  - Issue Date
- Students can view their certificates securely  
- Wallet connection via WalletConnect / Reown AppKit  
- Fully on-chain storage for certificate ownership  
- Open-source frontend with responsive design  
- Metadata stored on IPFS for decentralization
